.. _DESCRIPTION_SEC:

===========
Description
===========

.. epigraph::

   A logical, reasonably standardized, but flexible project structure for doing and sharing data science work.

   -- Data Science Team

Making your project **reproducible** is one of the key elements when
doing research in pretty much any field, including Data Science. This
package aims at making it easier to be transparent about what
*steps* and *commands* are used when producing tangible results for a
research project.

In the following sections, I will provide instructions on how to
to use this project tempalte, as well as how to make the most out
of this template.

This documentation is part of the repository
`cookiecutter-data-science-vc <https://github.com/vcalderon2009/cookiecutter-data-science-vc>`_,
and has been adapated from the `Cookiecutter Data Science Project template <https://drivendata.github.io/cookiecutter-data-science/>`_ 
by `Driven Data <https://www.drivendata.org/>`_ organization.
